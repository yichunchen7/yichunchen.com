Online Shopper's
Purchasing Intention

This project intended to build a machine learning model for an e-commerce company to predict if a website visitor will make a purchase based on their historical browsing data. After testing every available classifier in sklearn, we concluded that Random Forest Classifier was the winner, which was able to predict if a customer will make a purchase with 85% confidence , and if a customer won’t make a purchase with 86% confidence.

Technology used: Python (pandas, numpy, seaborn, matplotlib.pyplot, scikit-learn)
